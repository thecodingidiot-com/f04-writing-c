# f04 — Writing C

Companion repository for the **[f04 — Writing C](https://thecodingidiot.com/chapters/f04-writing-c)** chapter on [thecodingidiot.com](https://thecodingidiot.com).

---

## Follow my journey

You are working through the implementation pages. Copy the tester into your `f04-practice` directory and run it from there when you have finished.

```bash
git clone https://github.com/thecodingidiot-com/f04-writing-c.git
cp f04-writing-c/test.sh ~/f04-practice/
cd ~/f04-practice
bash test.sh
```

Use `bash test.sh --help` to see what each check verifies.

## Follow your journey

You built it independently. Copy the tester into your working directory and run it to check your solution against the project brief:

```bash
git clone https://github.com/thecodingidiot-com/f04-writing-c.git
cp f04-writing-c/test.sh ~/f04-practice/
cd ~/f04-practice
bash test.sh
```

The tester prints `PASS` or `FAIL` with a short explanation for each check.

---

## What the tester checks

1. `hello.c` compiles with `-Wall -Wextra` and prints a greeting.
2. `calculator.c` compiles and produces correct output for `+`, `-`, `*`, `/`, division by zero, and an unknown operator.
3. `words.c` compiles and correctly counts words and characters for several inputs.
4. `guess.c` compiles cleanly. The game is interactive — play it yourself.
